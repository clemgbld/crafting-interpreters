# Crafting interpreters

Repo containing the challenges as well as the code of the book crafting interpreters from Robert Nystrom.

## clox make file commands

### compile main source files (non-test)

```bash  
make all
```

### run main program

```bash
make run

```
### run main program with arguments

```bash
make run ARGS="file.txt"

```

### run test

``` bash
make test
```
